# gmail-filter-ui

ui for my gmail that helps me filter emails based on api calls to LLM models

UI Functionality
- List all emails
- Label each email: Spam, low prio, etc

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/gmail-filter-ui.git
cd gmail-filter-ui
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
